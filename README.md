# Список задач
## Задача 1

Краткое условие: "Напишите код, который вычисляет первые 100 чисел Фибоначчи."

Алгоритм решения: Создаем "базу" в виде 0 и 1, дальше на каджом шаге складываем сумму двух переменных и меняем их значения на новые.

## Задача 2

Краткое условие: "Марсоход получает с Земли сокращённые инструкции с заданиями. Число перед квадратными скобками обозначает, сколько раз надо повторить последовательность внутри скобок. Скобки могут быть и вложенными. Напишите программу, которая расшифровывает сжатые сообщения и возвращает строку с командами."

Алгоритм решения: Проходим по строке слева направо. Если встречаем цифру, то считываем всё число, сохраняем в стек текущий результат и это число. Если встречаем букву, то добавляем её к текущему результату. Если встречаем закрывающуюся скобку — извлекаем из стека предыдущий результат и число повторов, умножаем текущий результат на это число и добавляем к предыдущему. В конце возвращаем собранную строку.

## Задача 3

Краткое условие: "Фермер выращивает кусты малины, высаженные в один ряд.
Всего на участке растёт N кустов, каждый куст имеет свою урожайность.
Фермер решил выкопать часть кустов так, чтобы:
1.	Не осталось двух соседних выкопанных кустов.
2.	Суммарная урожайность выкопанных кустов была максимальной.

Выведите одно число — максимальную суммарную урожайность кустов, которые фермер может выкопать, не нарушая условие.
"

Алгоритм решения: Считываем количество кустов и их урожайность в список. Создаем массив dp, где dp[i] — максимальная сумма, которую можно собрать на первых i+1 кустах. Заполняем базу динамики, где dp[0] - урожайность первого куста, а dp[1] - максимум из урожайности первого и второго кустов. Для каждого следующего куста (от 2 до n-1) либо не берем текущий куст (берем предыдущий результат dp[i-1]), либо берем текущий куст и лучший результат до позапрошлого куста (dp[i-2] + bush[i]). Выбираем максимум из этих двух вариантов. Ответ храниться в последнем элементе dp.
